# This branch and all children don't have node modules
## Installing Node Modules
```sh
npm i
```
or
```sh
npm install
```

# Additional Libraries
- This project relies on bootstrap. To install bootstrap
```sh
npm install bootstrap
```

# Running the project
```sh
npm run
```
